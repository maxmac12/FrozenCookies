FrozenCookies

An automated tool for Cookie Clicker.

Bookmarklet Install

  1. Paste the contents (not the url) of https://maxmac12.github.io/FrozenCookies/fc_bookmarklet_loader.js into a bookmark.
  2. Load up Cookie Clicker.
  3. Load the recently created bookmark.

Userscript Install

    If you don't already have it, install the required browser add-on: Tampermonkey (Chrome) or Greasemonkey (Firefox)

  1. Go to https://maxmac12.github.io/FrozenCookies/fc_userscript_loader.user.js
  2. Click the 'install' button in the upper right corner.
  3. Load up Cookie Clicker.
